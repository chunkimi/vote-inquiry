# 2023 The F2E Mission 2 - 總統即時開票全台地圖

## 作品說明

透過圖表檢視市、區、里的歷史數據分布，畫面依使用者友善

## 系統說明

- npm install - 安裝套件
- npm run dev - 運行 dev 環境

## Node.js version

v18.18.2

## 資料夾說明

- assets - 圖片放置處
- components - 元件放置處
- styles - style 設定
- views - 畫面放置處
